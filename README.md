# Boulder Fit

Personal bouldering + fitness dashboard. Single static HTML file with local-storage persistence.

## Local use

Just open `index.html` in any browser.

## Deploy

Deployed to Railway as a static site served by [`serve`](https://www.npmjs.com/package/serve).

```bash
npm install
npm start
```

Railway auto-detects Node, installs deps, and runs `npm start`.
